# 22-sakamoto.github.io

就職活動向けのシンプルなポートフォリオサイトです。GitHub Pagesで公開する前提の、静的なHTML/CSS構成です。

## ファイル構成

- `index.html`: ページ本文
- `styles.css`: レイアウトとデザイン
- `assets/portfolio-hero.webp`: ページで使う軽量ヒーロー画像
- `assets/portfolio-hero.png`: WebP非対応環境向けの画像

## 先に差し替える場所

- `index.html` の `Sakamoto`: 表示名
- `https://github.com/22-sakamoto`: GitHubプロフィールURL
- `学歴` セクション: 学校名、学部学科、卒業予定年月
- `所属` セクション: ゼミ、サークル、団体、役割
- `使用経験のあるツール・スキル` セクション: 技術、ツール、経験内容
- `成果物` セクション: 実際の制作物、担当範囲、使用技術、リンク
- `Contact` セクション: メール、履歴書PDF、Wantedlyなどの連絡先

## 公開メモ

このリポジトリは `22-sakamoto.github.io` というGitHub Pages用の名前なので、通常は `main` ブランチへpushすると公開できます。
表示されない場合は、GitHubのリポジトリ設定で Pages の公開元が有効になっているか確認してください。
